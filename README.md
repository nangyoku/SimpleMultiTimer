# SimpleMultiTimer
Simple Multi Timer and Stopwatch built with vanilla JavaScript.

Run:

- Open [SimpleMultiTimer/index.html](SimpleMultiTimer/index.html) in your browser.

Usage:

- Timers: enter an optional name, minutes and seconds, then click **Add timer**.
- Stopwatches: enter an optional name and click **Add stopwatch**.
- Each item can be started, paused, reset, or removed.
- Timers play a short beep when finished (if the browser allows audio).

Pomodoro:

- Use the **Pomodoro** panel to start a 25-minute work session by clicking **Start**.
- After a work session ends the app automatically switches to a short break (5 min) or a long break (15 min) every 4 cycles.
- You can manually start a short or long break using the **Short Break** or **Long Break** buttons.
- Use **Pause** and **Reset** to control the pomodoro cycle.
